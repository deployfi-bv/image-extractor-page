Method 4: Add a Watermark

How it Works

Watermarking involves adding a visible logo, text, or pattern over your image. This doesn't technically stop anyone from downloading the image, but it makes it less appealing for them to use it, as your brand or name is attached.

You can add a watermark using image editing software (like Photoshop or GIMP) before you upload the image. Alternatively, you can add it on-the-fly using HTML and CSS, though this is less secure.

Benefit: If your watermarked image is shared, it can act as a form of marketing for your brand.

Example

Here is an image with a simple text watermark added using HTML and CSS.

Your Brand ©

The Code (HTML/CSS Method)

To create a simple watermark with CSS, you wrap the image in a container and add another element for the watermark text.

<div class="watermarked-container">
    <img src="your-image.jpg" alt="My Image">
    <div class="watermark">Your Brand ©</div>
</div>

Then, use CSS to style and position the watermark.

.watermarked-container {
    position: relative;
    display: inline-block;
}

.watermark {
    position: absolute;
    bottom: 20px;
    right: 20px;
    color: white;
    background-color: rgba(0, 0, 0, 0.5);
    padding: 5px 10px;
    border-radius: 5px;
}
