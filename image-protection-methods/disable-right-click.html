Method 1: Disable Right-Click

How it Works

This method uses a small piece of JavaScript to prevent the browser's context menu (the menu that appears when you right-click) from opening. When a user tries to right-click on an image to save it, nothing will happen.

Limitation: This is a very basic protection. It won't stop users who know how to use browser developer tools to find the image source directly.

Example

Here is an image with right-click disabled. Try right-clicking on it.

The Code

To use this method, add the oncontextmenu="return false;" attribute to your image tag.

<img src="your-image.jpg" oncontextmenu="return false;">
