Method 2: Use a Transparent Overlay

How it Works

This method involves placing a transparent element (like a `<div>` or even a transparent GIF image) directly on top of your actual image. When a user right-clicks and selects "Save Image As...", they will save the transparent overlay instead of the image underneath.

Limitation: Like the right-click disabling method, this can be bypassed by users who inspect the page's HTML and find the underlying `<img>` tag.

Example

Try to save the image below. You will likely save a transparent file.

The Code

You need both HTML and CSS. First, wrap your image in a container and add an overlay `<div>`.

<div class="image-container">
    <img src="your-image.jpg" alt="My Image">
    <div class="overlay"></div>
</div>

Then, add the following CSS to position the overlay on top of the image.

.image-container {
    position: relative;
    display: inline-block;
}

.overlay {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    z-index: 10;
}
