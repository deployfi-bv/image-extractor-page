Method 3: Use CSS Background Images

How it Works

Instead of using an `<img>` tag, you can set your image as the background of an element like a `<div>`. Because the image is part of the CSS, there is no direct "Save Image As..." option when a user right-clicks it. The user would have to inspect the CSS code to find the image URL.

Limitation: This method is not ideal for all images, especially those that are critical to your content (like product photos). It's best suited for decorative images. Also, like other methods, it can be bypassed using browser developer tools.

Example

The image below is a `<div>` with a background image. Try right-clicking it.

The Code

First, create a `<div>` in your HTML.

<div class="image-as-background"></div>

Then, use CSS to set the background image and dimensions.

.image-as-background {
    width: 100%;
    height: 400px; /* An element with a background image must have a size */
    background-image: url('your-image.jpg');
    background-size: cover;
    background-position: center;
}
